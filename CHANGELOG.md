# CHANGELOG

## [2.27.0-L1] - 2026-07-11

Первый релиз под именем **WorldPainter Languages** (ранее WorldPainter RU) на базе WorldPainter 2.27.0.

### Добавлено

- Украинская (uk), белорусская (be) и казахская (kk) локализации в полном паритете с русской.
- Разделение переводов на доменные bundle-файлы: strings, blocks, gamedata, layers, swing.
- Маршрутизация ключей по префиксу в WPI18n с фолбэком на основной bundle.
- Просьба поставить звёздочку репозиторию в диалоге поддержки.

### Изменено
- Настройки теперь хранятся в отдельной папке `WorldPainter Languages` (на Windows — `%APPDATA%\WorldPainter Languages`); при первом запуске существующие настройки автоматически копируются из старой папки `WorldPainter` (сама она не трогается).

- Проект переименован в WorldPainter Languages.
- Диалог поддержки обновлён под новый бренд; всплывающие сообщения диалога локализованы.
- Все .properties приведены к чистому UTF-8 (без \uXXXX-эскейпов), CRLF.
- Финальная вычитка переводов RU/UK/BE/KK.

### Удалено

- 195 неиспользуемых ключей локализации (легаси и осиротевшие), синхронно во всех языках.
- Внутренние инструменты и черновые документы из дерева исходников.

## [2.27.0-r1] - 2026-07-06

Первый публичный релиз WorldPainter RU на базе WorldPainter 2.27.0.

### Добавлено

- Русская локализация интерфейса.
- Русские resource bundles для основных строк и enum-значений.
- Локализация стандартных Swing/JOptionPane/JFileChooser строк.
- Документация для публикации на GitHub.
- Release checklist и packaging notes.
- Скрипт `scripts/build-windows-installer.ps1` для сборки Windows installer через `jpackage`.
- Portable layout и release assets для GitHub Releases.

### Изменено

- Пользовательские строки во многих UI-классах переведены на существующую систему ResourceBundle.
- Диалоги, progress-сообщения, import/export/merge/startup/recovery тексты подготовлены для русской локализации.
- Доработаны installer metadata, иконка, portable layout и структура release assets.
- Диалог поддержки приведён к более компактному размеру и дополнен ссылкой на проект перевода.

### Не изменялось

- Алгоритмы генерации миров.
- Форматы файлов.
- Maven-структура проекта.
- Бизнес-логика WorldPainter.
- UI-архитектура и внешний вид.

### Известные ограничения

- Не все редкие аварийные окна можно подтвердить без ручного GUI-тестирования.
- Стороннее предупреждение JIDE о лицензии не относится к русификации и требует отдельного решения.
